Research-Inspired Extensions in Visual Studio Gallery

There are TONS of extensions in the Visual Studio App Store (i.e., Visual Studio Gallery), most of which are small improvements on VS's default tooling and, in my opinion, not worth the hassle of installing. However, there's a new class of extensions, inspired by years of software engineering research, that show real potential to enhance developers' performance. In this talk I'll present one commercial extension, one free open source extension, and two free closed extensions, all inspired by research. I'll discuss how developers can use these tools to dramatically improve their performance on common development tasks, backed by empirical data and user studies. By the end of this talk I expect each developer to have found tool support to remove at least one major pain point from their day. http://www.codeproject.com/Articles/665524/Research-Inspired-Extensions-Hit-Visual-Studio-App

David Shepherd

David Shepherd is a software tools researcher, turned professional tools developer, turned software tools researcher and open source contributor. Inspired by Andi Marcus's research on "Google-ing source code" he's recently spent his time developing Sando, a free open source search extension for Visual Studio with over 6000 downloads. You'll also find him evangelizing the merits of better code search tools at conferences like devLink. He lives and works in the RTP area where he leads ABB's Software Engineering Research Group.
