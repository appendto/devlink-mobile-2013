Building a Windows CI & CD Environment

This fast-paced session will use an OpenStack cloud to spin up Windows servers and create a Continuous Integration/Continuous Deployment (CI/CD) environment. We'll do this in one hour. Seriously. Using open source software, and github as our source repository, you'll learn how to create a Windows development environment where you need just one click to build your software, build the tests, execute the tests and deploy to a web server. For this demonstration, we'll be using Rackspace's OpenStack implementation for our servers, but you can use any cloud in your implementation. Since OpenStack is open source, you can even build your own cloud for use in your own CI/CD setup.

Don Schenck

Don Schenck is a Rackspace OpenStack .NET SDK Developer Advocate – meaning he writes code, writes about code, speaks at conferences and helps customers. Prior to Rackspace, Don worked across a broad range of industries, from helping pharmaceutical companies track and improve quality to building software to control machines that cut and bend reinforcing steel. Most recently, Don was involved in writing mobile and kiosk applications. When he’s not coding or writing or speaking at a conference, he prefers to surf (waves, not the internet), lift weights and whine about things to his longsuffering wife, Patti.
