Mastering asynchronous flow control in JavaScript (or how to avoid callback hell)

Asynchronous JavaScript can be very powerful, but also very messy if pains are not taken to tame callback spaghetti. This talk will cover a number of techniques like promises, the async library, and messaging that allow developers to manage the madness. I will also discuss the JavaScript event model and why async functions are critical in the single-threaded node.js environment.

Async Intermediate Javascript

Nicholas Cloud

For the last decade Nicholas has worked hard to turn his skills into a career. He has developed solutions on both Linux and Windows platforms, programming professionally in JavaScript, C#, Ruby, and PHP. He has extensive experience with web technologies, web services, and Windows desktop development. Nicholas has used more open source libraries than you can shake a stick at, and has contributed to a few as well. He is the founding member of the St. Louis ALT .NET user group, speaks at a variety of user groups and conferences, and writes technical articles and blog posts in his spare time. Nicholas has a real passion for deconstructing business problems, creating abstract, reusable and flexible solutions, and creating awesome software that rocks.
