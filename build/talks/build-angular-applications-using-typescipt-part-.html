Build Angular applications using TypeScipt - Part 2

Wednesday
4:00 PM – 5:15 PM
Meeting Room 5

In this session we will build on knowledge acquired in Build Angular application using TypeScipt and Twitter Bootstrap - Part 1. We will build a small application step by step, using ASP.NET MVC to create a shell page. We will create Web Api classes using Entity Framework code first to supply the data. We will create views for our UI, services, controllers and widgets. We will see how Angular navigation works. We will unit test(s) for our TypeScript code. Learning objectives 1. See how to build an Angular JS based application with TypeScript step by step 2. See Angular components in action. 3. Understand testing options with Jasmine

ASP.NET MVC Angular JavScript TypeScript Beginner

Sergey Barskiy

Sergey Barskiy is an architect with Tyler Technologies. He is a Microsoft MVP. He holds the following certifications: MCPD, MCTS, MCSD for .NET, MCAD for .NET, MCDBA, MCP. He has been working with Microsoft technologies for 15 years. He is a frequent speaker at various regional and national conferences, such as VS Live, DevLink, CodeStock, and Atlanta Code Camp as well as local user groups. He is one of the organizers of Atlanta Code Camp and Atlanta Windows Apps users group. He is also INETA membership mentor for the state of GA.
