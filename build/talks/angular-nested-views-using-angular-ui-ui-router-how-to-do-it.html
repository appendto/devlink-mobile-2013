Angular Nested Views using angular-ui / ui-router : How to do it

Angular built in routing provides a relatively simple way to handle basic simple routing, but does not explicitly cater for nested views very well. If you need to do more complex scenarios like composite apps with multiple nested and sibling views, the built in routing starts to falter and you will probably land up developing something custom. Angular-ui/ui-router is ngRoute on steroids and provides a great way to address these sorts of problems. Come and kick the tires and take it out for a sprint or two. You will come away with a clear understanding of how this angular module can work for you and how to use it.

Angular Javascript Asp.net Intermediate Mvc Ui-router
