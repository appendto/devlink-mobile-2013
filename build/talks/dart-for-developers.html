Dart for Developers

Thursday
4:00 PM – 5:15 PM
Meeting Room 13

So maybe you’ve tried CoffeeScript or a framework that is supposed to make JavaScript development more like that of object oriented or functional languages. Maybe you’ve been frustrated by the lack of good development tools for JavaScript. Maybe you’re tired of using several languages to develop a web application. Dart is a new platform for web development that answers all of these problems. Taking cues from the trends in modern languages, it seeks to create a familiar yet flexible environment without sacrificing performance. But Dart is more than a language, it’s a platform with tools, libraries and runtimes for writing end-to-end web applications. It can also be compiled to JavaScript so you can start using it today in modern browsers across devices from desktop to mobile.

Douglas Starnes

Douglas Starnes is a polyglot ninja in the Memphis, TN area. He has spent the past 7 years developing software in multiple languages including C#, Python, JavaScript and more on mobile, cloud and web platforms. His community involvement includes speaking at conferences such as DevLink, St. Louis Days of .NET, CodePaLOUsa, CodeStock and PyOhio in addition to the .NET and Java User Groups in Memphis. He is the co-organizer of the Memphis Python User Group (MEMPy) and one of the organizers of PyTennessee, the annual state of Tennessee Python conference. Douglas is also a founding board member of GeekMemphis, the local technology advisory organization in Memphis. He can be found on the web at http://douglasstarnes.com and on Twitter at @poweredbyaltnet.
