ReactJS - Rethinking Client Side UI

ReactJS has a fascinatingly different take on how to manage client-side UI. Born out of collaboration between Facebook and Instagram, ReactJS focuses on view-related concerns – while letting you choose the right tool(s) for other client side needs – building components from JavaScript, JSX (a markup syntax) and CSS. We’ll explore the basics of how to create components in React, how React manages to be so fast (internally rendering to an intermediate DOM to perform a diff and 'patching' the actual DOM with changes), how to use React along side other great client-side tools, and more.
