Aah I get it ! Responsive design using Angular made easy

Friday
1:00 PM – 2:15 PM
Meeting Room E

How does responsive design work ? How can you do responsive design using the popular angular framework ? In this interesting session we will go though a complete responsive design example and show you an easy way to do it. You will learn how to bring css3 transitions, animations and media queries together using angular.

Angular CSS3 Javascript Responsive Design Intermediate

Mitch Harpur

I am a senior consultant with Wintellect and and an instructor with WintellectNow, an on demand training resource. For close to 20 years I have been developing all sorts of things, so I have been around the block once or twice with several technology stacks, but I continue to love to learn new tricks. I am an Electrical Engineer by profession, but I love and extensively use the technologies that make up the modern web experience today, from the mobile experience in the cloud to the intranet in the enterprise .
