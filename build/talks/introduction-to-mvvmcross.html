Introduction to MvvmCross

Thursday
9:30 AM – 10:45 AM
Meeting Room 7/8

In this course we'll learn how to build an application using MvvmCross, a cross platform Mvvm C# framework for Xamarin.iOS, Xamarin.Android, and Windows Phone. We'll focus on the essential pieces you'll need to know to get up and running with MvvmCross quickly. By the end of this course you'll have experience with building mobile applications for iOS, Android, and Windows Phone using MvvmCross.

CrossPlatform Xamarin Intermediate

Peter Mourfield

Peter Mourfield has been developing software for 20 years and currently works for TaxSlayer.com, an online income tax preparation web site. He lead the Augusta Developers Guild, and is an ASP Insider. Peter also holds a number of certifications including; Microsoft Certified Professional Developer (MCPD), Microsoft Certified Trainer (MCT), and Certified SCRUM Master.
