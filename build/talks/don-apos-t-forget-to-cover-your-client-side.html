Don't forget to cover your client side

Unit testing is an important step in the development workflow. It allows you to ensure the behavior of your application. Client side testing is especially important because when working with a dynamic language such as JavaScript it can give you an extra layer of protection against bugs. There are a lot of great libraries out there for client side testing. This talk will cover how to write client side unit tests, how to check for code coverage, and how to run tests on the maintainability of your client side code as well.
