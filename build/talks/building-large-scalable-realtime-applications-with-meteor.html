Building Large Scalable Realtime Applications With Meteor

Wednesday
10:15 AM – 11:30 AM
Meeting Room 7/8

Meteor is an amazing front-end and back-end framework that you just have to see to believe. This talk will focus on how to build a large scalable Meteor application complete with a capable test suite, architectural best practices and tips and tricks to get the best performance out of your application. If you have ever wanted to build a realtime application either for the web, or to package with Phonegap and distribute to native devices, this is a talk you can't afford to miss.

JavaScript Meteor MeteorJS Node Realtime Application Advanced

Tyson Cadenhead

Tyson is a JavaScript Engineer who specializes in crafting large scalable applications using the latest technologies. When he isn’t writing code, Tyson enjoys playing guitar, filmmaking, animation, illustration, blogging, gardening, reading books about theology, economics and philosophy, and spending time with his family
