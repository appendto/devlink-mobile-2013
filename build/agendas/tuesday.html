  • 9:00 AM – 5:00 PM

  • AngularJS Fundamentals

    AngularJS is a one of the most popular MVC frameworks in building a front-end application, and the best place to start learning about it is at the beginning. In this class we’ll lay the groundwork for learning AngularJS by introducing scope for data-binding, controllers for MVC on the client, events, filters to format output, directives to extend the HTML vocabulary, modules to group code, services to provide application-wide functionality, and routes to enable Single Page Applications (SPA).
      • TOPICS COVERED
      • Introduction to AngularJS
      • Controllers and Markup
      • Creating and Using Services
      • Routing
      • Creating Custom Directives
    • Workshop is hands-on, laptops are recommended
      Presented by

  • Getting Started with Microsoft Azure

    Microsoft is committed to moving forward in a mobile-first cloud-first world and Microsoft Azure is a big part of that commitment. Microsoft Azure is a set of independent services used to create secure, reliable, scalable, and highly available applications in the cloud. This class introduces the various components and services explaining what they each do and how they relate to one another. We will cover Platform as a Service (PaaS) and Infrastructure as a Service (IaaS) as well as how the two can interact and integrate with one another. This class will take you from not knowing anything about Microsoft Azure to being able to confidently develop and deploy applications on the platform.
      • TOPICS COVERED
      • Azure Management Dashboard and Developer Tools
      • Azure Websites
      • Cloud Services
      • Mobile Services
      • Azure Data Storage Options
      • Virtual Machines
      • Other Software Services (Active Directory, Authentication, etc)
    • Workshop is hands-on, laptops are recommended
      Presented by

  • JavaScript and jQuery Foundations

    A solid introduction to the core concepts of jQuery-focused front-end development that shows you how to interact with and manipulate sites. By the end of the day you’ll feel more confident about using jQuery and be ready to start developing for the front-end. During the workshop we’ll cover both the basics and some intermediates of the JavaScript programming language, as well as the jQuery JavaScript library.
      • TOPICS COVERED
      • JavaScript 101
      • jQuery Intro and Selections
      • jQuery Methods and Chaining
      • jQuery each, Deeper into Functions
      • jQuery and DOM Manipulation
      • Document.Ready
      • Events
      • Ajax
    • Workshop is hands-on, laptops are recommended
      Presented by

  • Web Developers Workflow

    Creating a website today requires lots of tools. These tools are becoming increasingly common in web development and can be used to help you in any web project of your own. Tools like Git for version control, Sass for extending CSS, JSHint for detecting errors or potential problems with your code, UglifyJS to minify and compress your code, Bower to manage your front-end dependencies, QUnit to run tests against your code and GruntJS to run all these tasks automatically and together at the same time.
    In this class, attendees will learn the foundations for implementing web projects. We’ll show you how to use each of these tools and then how to tie them all together with GruntJS to make managing the tasks that each of these tools need a breeze.
      • TOPICS COVERED
      • Understanding Git
      • Sass
      • JSHint/JSLint
      • UglifyJS
      • Bower
      • Qunit
      • GruntJS
    • Workshop is hands-on, laptops are recommended
      Presented by
