Git your .NET On!

  • beginner Git ALM
People generally don't think of version control systems as cool. That was before Git came along. Git is a distributed version control system that gives developer a level of power and flexibility that they haven't experienced before. Previously the darling of the open source community, Microsoft has recently embraced Git and integrated it into TFS. But what is Git and as a .NET developer how will your experience be different? This session will explain how Gits distributed nature differs from other version control systems. You will see how to use both the command line tools and the new TFS interface. The concept of a "Git workflow" will be explained and you will see how Git is used in many common scenarios. The bottom line is that Git lets you work how you want to work, and this session will get you started!
